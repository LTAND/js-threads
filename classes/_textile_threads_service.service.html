createThread

  • createThread(id: ThreadID, opts?: KeyOptions): Promise<ThreadInfo>
    • Defined in packages/service/src/service.ts:53

    createThread with id. This will overwrite any existing thread information.

    Parameters

    • id: ThreadID

      The Thread id.

    • Default value opts: KeyOptions = {}

      The set of keys to use when creating the Thread. All keys are "optional", though if no replicator key is provided, one will be created. Similarly, if no log key is provided, then the "host" Peer ID will be used. If no ReadKey is provided, the service will be unable to write records (but it may be able to return records).

    Returns Promise<ThreadInfo>
