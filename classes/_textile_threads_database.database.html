open

  • open(options?: StartOptions): Promise<void>
    • Defined in database/src/db.ts:188

    Open the database. Opens the underlying datastore if not already open, and enables the dispatcher and underlying services (event bus, network network, etc). If threadID is undefined, and the database was already bootstrapped on a thread, it will continue using that thread. In the opposite case, it will create a new thread. If threadID is provided, and the database was not bootstrapped on an existing thread, it will attempt to use the provided threadID, otherwise, a thread id mismatch error is thrown.

    Parameters

    • Default value options: StartOptions = {}

      A set of options to configure the setup and usage of the underlying database.

    Returns Promise<void>

Static fromAddress

  • fromAddress(addr: Multiaddr, datastore: Datastore<any>, threadKey?: ThreadKey, options?: InitOptions): Promise<Database>
    • Defined in database/src/db.ts:121

    fromAddress creates a new database from a thread hosted by another peer. Unlike the Database constructor, this will auto-start the database and begin pulling from the underlying Thread.

    Parameters

    • addr: Multiaddr

      The address for the thread with which to connect. Should be of the form /ip4/<url/ip-address>/tcp//p2p//thread/

    • datastore: Datastore<any>

      The primary datastore.

    • Optional threadKey: ThreadKey

      Set of symmetric keys.

    • Default value options: InitOptions = {}

      A set of database options.

    Returns Promise<Database>
