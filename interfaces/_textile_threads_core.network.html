createThread

  • createThread(id: ThreadID, opts: KeyOptions): Promise<ThreadInfo>
    • Defined in core/src/network/network.ts:28

    createThread with id.

    Parameters

    • id: ThreadID

      The Thread id.

    • opts: KeyOptions

      The set of keys to use when creating the Thread. All keys are "optional", though if no replicator key is provided, one will be created (and returned). Similarly, if no LogKey is provided, then a private key will be generated (and returned). If no ReadKey is provided, the network will be unable to write records (but it may be able to return records).

    Returns Promise<ThreadInfo>
