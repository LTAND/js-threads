readLock

  • readLock(key: Key, timeout?: undefined | number): Promise<void>
    • Defined in packages/store/src/stores/store.ts:118

    Acquire a read lock on a given key. The datastore is only allowed to acquire a lock for keys it 'owns' (any decedents of its prefix key).

    Parameters

    • key: Key

      The key to lock for reading.

    • Optional timeout: undefined | number

      How long to wait to acquire the lock before rejecting the promise, in milliseconds. If timeout is not in range 0 <= timeout < Infinity, it will wait indefinitely.

    Returns Promise<void>
