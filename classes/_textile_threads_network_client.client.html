Class Client

Client is a web-gRPC wrapper client for communicating with a webgRPC-enabled Textile server. This client library can be used to interact with a local or remote Threads gRPC Network.

createThread

  • createThread(id: ThreadID, opts: NewThreadOptions): Promise<ThreadInfo>
    • Defined in network-client/src/index.ts:162

    createThread with id.

    Parameters

    • id: ThreadID

      The Thread id.

    • opts: NewThreadOptions

      The set of keys to use when creating the Thread. All keys are "optional", though if no replicator key is provided, one will be created (and returned) on the remote network. Similarly, if no LogKey is provided, then a private key will be generated (and returned) on the remote network. If no ReadKey is provided, the remote network will be unable to write records (but it can return records).

    Returns Promise<ThreadInfo>
