encodeBlock

  • encodeBlock(block: Block, keyiv: Uint8Array, opts?: Options): Block<Buffer>
    • Defined in packages/encoding/src/coding.ts:20

    EncodeBlock returns a node by encrypting the block's raw bytes with key.

    Parameters

    • block: Block

      The IPLD block to encrypt and encode.

    • keyiv: Uint8Array

      A random key of length 44 bytes (32 bytes for key and 12 bytes for iv) to use for encryption.

    • Default value opts: Options = defaultOptions

      Options to control encoding and encryption. Defaults to CBOR encoding with the SHA256 hash function, and AES GCM 256-bit encryption.

    Returns Block<Buffer>
