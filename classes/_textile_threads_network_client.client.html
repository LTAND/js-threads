Class Client

Client is a web-gRPC wrapper client for communicating with a webgRPC-enabled Textile server. This client library can be used to interact with a local or remote Threads gRPC Network.

createThread

  • createThread(id: ThreadID, opts: NewThreadOptions, ctx?: Context): Promise<ThreadInfo>
    • Defined in network-client/src/index.ts:180

    createThread with id.

    Parameters

    • id: ThreadID

      The Thread id.

    • opts: NewThreadOptions

      The set of keys to use when creating the Thread. All keys are "optional", though if no replicator key is provided, one will be created (and returned) on the remote network. Similarly, if no LogKey is provided, then a private key will be generated (and returned) on the remote network. If no ReadKey is provided, the remote network will be unable to write records (but it can return records).

    • Optional ctx: Context

      Context object containing web-gRPC headers and settings.

    Returns Promise<ThreadInfo>

getToken

  • getToken(identity?: Identity, ctx?: Context): Promise<string>
    • Defined in network-client/src/index.ts:114

    Obtain a token for interacting with the remote network API.

    note

    If an identity is not provided, a random PKI identity is used. This might not be what you want! It is not easy/possible to migrate identities after the fact. Please supply an identity argument if you wish to persist/retrieve user data later.

    Parameters

    • Optional identity: Identity

      The generic identity to use for signing and validation.

    • Optional ctx: Context

      Context object containing web-gRPC headers and settings.

    Returns Promise<string>
