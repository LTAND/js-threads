Class Dispatcher

Dispatcher is used to dispatch events to registered reducers. This is different from generic pub-sub systems because reducers are not subscribed to particular events. Every event is dispatched to every registered reducer. Dispatcher is based on the singleton dispatcher utilized in the "Flux" pattern (see https://github.com/facebook/flux).
